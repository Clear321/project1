##### 选做内容（建议选做）： 基于数字滤波器的 ECG 信号降噪处理

基于 MATLAB 的数字滤波器设计方法，设计数字滤波器用于心电图
（Electrocardiogram, ECG）信号的降噪处理。针对 ECG 信号中常见的高斯白噪
声、工频干扰及基线漂移等噪声类型，分别设计 IIR 与 FIR 数字滤波器（或者小
波降噪处理），以有效抑制噪声干扰，提升信号质量与可靠性，为后续心电信号
分析与诊断提供清晰、稳定的数据基础。

ECG 信号与噪声信号生成
在 MATLAB 中导入 ECG 信号，采样率选取 500Hz。第一组在 ECG 信号中
加入信噪比为 10dB 的高斯白噪声；第二组在 ECG 信号中叠加 50Hz 工频干扰的
信号，该干扰信号主要是由电力系统的频率干扰和各种谐波干扰引起的高频噪声；
第三组引入基线漂移的干扰信号，基线漂移信号属于低频干扰，呼吸的节奏、四
肢动作等都有可能造成基线漂移。绘制 ECG 信号与加入三种噪声信号后的时域
波形对比，然后通过对四组信号进行 FFT，绘制信号的幅度谱。


高斯白噪声的滤除


分别使用低通 IIR 滤波器和低通 FIR 滤波器滤除 ECG 信号中的高斯白噪声。
由于 ECG 信号有效成分集中于 0-40Hz，而高斯白噪声具有全频带均匀分布特性，
因此设定 25Hz 通带截止频率与 40Hz 阻带截止频率。Butterworth 低通滤波器指
标为： _[f]_ _p_ [] 25 Hz， _[f]_ _s_ [] 40 Hz， _A_ P  1 dB， _[A]_ _s_ [] 15 dB，Fs=500Hz。绘制滤波器

的滤波效果图。


工频干扰噪声的滤除


使用 IIR 带阻滤波器消除掉 ECG 信号中的工频干扰信号，设计带阻滤波器
参数指标为： _[f]_ _p_ [] [45,55] Hz， _[f]_ _s_ [] [48,52] Hz， _A_ P  1 dB， _[A]_ _s_ [] 15 dB。绘制滤波

器的滤波效果图。


基线漂移噪声的滤除


使用高通 IIR 滤波器和高通 FIR 滤波器滤除 ECG 信号中的基线漂移噪声信
号。由于基线漂移具有极低的频率特性，设置高通滤波器参数指标为： _[f]_ _p_ [] 0.7 Hz，

_[f]_ _s_ [] 0.3 Hz， _A_ P  1 dB， _[A]_ _s_ [] 15 dB。在消除低频率分量噪声的同时，保留 0.7Hz
以上的 ECG 信号特征，绘制滤波器的滤波效果图。

clear; clc;

% 导入 ECG 信号

val = importdata('Ecg.txt'); % fs=500Hz

signal = val(1,1:1800);

t0 = 1:length(signal);

fs = 500;

% 时域波形绘制

figure(1)

subplot(4,1,1);

plot(t0/100,signal,'k');

title('干净的 ECG 信号');

xlabel('时间/s');

ylabel('幅值');

grid on;

% 加入高斯白噪声

Signal1 = awgn(signal,10,'measured'); % 10dB

subplot(4,1,2);

plot(t0/100,Signal1,'k');

title('加入高斯白噪声的信号');

xlabel('时间/s');

ylabel('幅值');

grid on;

% 加入工频干扰

av = 100; % 幅值 AV

f0 = 50;

t = 1:length(signal);

noise2 = av * cos(2*pi*f0*t/fs);

Signal2 = noise2 + signal;

subplot(4,1,3);

plot(t0/100,Signal2,'k');

title('加入工频干扰的信号');

xlabel('时间/s');

ylabel('幅值');

grid on;

% 加入基线漂移

n1 = length(signal)/3;

x1 = zeros(1,n1);

noise3 = [x1, x2];

Signal3 = noise3 + signal;

subplot(4,1,4);

plot(t0/100,Signal3,'k');

title('加入基线漂移的信号');

xlabel('时间/s');

ylabel('幅值');

grid on;
